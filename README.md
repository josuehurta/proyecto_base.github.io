# Spring Thymeleaf Demo

Este es un proyecto de demostración utilizando Spring Boot y Thymeleaf.

## Requisitos

- JDK 8 o superior
- Maven
- IDE (IntelliJ IDEA, Eclipse, Visual Studio Code, etc.)

## Cómo ejecutar
1. **Clona el repositorio**:
    ```sh
    git clone <URL del repositorio>
    ```

2. **Navega al directorio del proyecto**:
    ```sh
    cd sfg-thymeleaf-course-demo
    ```

3. **Compila y ejecuta la aplicación**:
    ```sh
    mvn spring-boot:run
    ```

4. **Abre tu navegador y visita**:
    ```sh
    http://localhost:8080/product/1
    ```

## Estructura del proyecto

- `src/main/java`: Código fuente de Java.
- `src/main/resources`: Recursos del proyecto, incluyendo plantillas Thymeleaf.
- `src/test`: Pruebas unitarias.
